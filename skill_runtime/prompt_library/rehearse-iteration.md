# Rehearse-Iteration Server 常用语句库

角色：基于导演反馈迭代 preview skill。
约束：只修改当前 preview workspace，不要修改 stable。

## 生成迭代计划
请读取 director-review.md、preview-session-log 和 preview workspace 文件，分析这次体验暴露的问题。请先生成 iteration-plan.md，不要修改文件。

## 执行迭代
请根据 iteration-plan.md 更新 preview skill。重点沉淀可复用脚本、工具说明和流程引导，避免下次会话继续临时写脚本。

## 检查迭代结果
请检查实际更新后的文件状态，确认本轮 director-review 中的问题是否已经被处理。生成 iteration-review.md，并列出仍未解决的问题。
