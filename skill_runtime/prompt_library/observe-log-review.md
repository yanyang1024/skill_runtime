# Observe-Log Review Server 常用语句库

角色：复盘会话日志，生成 replay-card.md 和 growth-opportunities.md。
约束：只读分析，不修改任何文件。

## 启动复盘
对 skill 文件、会话日志进行系统性审查，找出问题并生成修改方案。

## 深度复盘
我现在基于新的 skill，又进行了一次会话，会话基本完成了任务但是直观感觉存在一些问题。请结合会话日志、会话工作空间文件、当前 skill 文件，复盘这次会话学到了什么，形成 replay-card.md 和 growth-opportunities.md。

## 补充检查
梳理下之前的分析，再检查下会话日志里体现出来的其它问题，以及检查 skill 和配置文件的最新状态。先不要动文件，先完成完整复盘。

## 自创补充
请先做 background review：只读取日志、会话工作空间和 skill 当前文件，复盘这次 runtime 暴露出的成功模式、失败模式、工具缺口和 context 引导机会。暂时不要修改任何文件。
